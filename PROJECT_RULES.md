# PROJECT RULES

This project is a premium Thailand visa platform focused on clarity, trust, accessibility, and simplicity.

The website should feel:
- modern
- calm
- spacious
- trustworthy
- efficient
- highly usable
- premium without luxury intimidation

Reference aesthetic:
- Bilt
- Stripe
- Mercury
- Linear
- modern fintech UX

Avoid:
- clutter
- gradients everywhere
- startup clichés
- excessive animations
- glassmorphism overuse
- generic AI-generated layouts
- oversized hero sections
- unnecessary decorative elements

Design principles:
- accessibility first
- mobile-first responsive design
- restrained premium aesthetic
- generous whitespace
- clear information hierarchy
- extremely readable typography
- fast loading performance
- reusable section architecture

Frontend stack:
- Next.js App Router
- Tailwind CSS
- shadcn/ui
- Radix UI
- Framer Motion

All components should:
- be reusable
- be modular
- support responsive layouts
- avoid unnecessary complexity

SEO is critical:
- semantic HTML
- proper heading structure
- internal linking
- fast page speed
- structured content hierarchy

The website must immediately communicate within 5 seconds:
- Thailand visa services
- simplicity
- trust
- fast communication
- high review credibility
- visa support expertise

Primary conversion goals:
- LINE contact
- WhatsApp contact
- inquiry form submissions

Google reviews are extremely important trust elements throughout the site.
