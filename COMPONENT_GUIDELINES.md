# COMPONENT GUIDELINES

All sections and components should be reusable and modular.

Preferred architecture:
- one section = one component
- small reusable cards
- clean layout wrappers
- centralized spacing consistency

Section spacing should feel:
- spacious
- premium
- calm
- readable

Avoid cramped layouts.

Preferred section structure:
- section label
- strong headline
- concise supporting copy
- clean CTA
- optional trust proof

Typography rules:
- large bold headlines
- restrained paragraph widths
- strong visual hierarchy
- avoid excessive font sizes on mobile

Buttons:
- rounded but not pill-shaped
- minimal
- high contrast
- clear CTA hierarchy

Animations:
- subtle only
- fade
- slight motion
- no excessive movement

Cards:
- soft borders
- minimal shadows
- clean padding
- no noisy UI

Mobile UX is extremely important.

All layouts should:
- stack beautifully
- preserve spacing
- maintain readability
- avoid oversized text blocks

Every section should feel intentional and premium.
