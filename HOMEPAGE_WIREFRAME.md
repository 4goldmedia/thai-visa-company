# HOMEPAGE WIREFRAME

## Homepage Goal

The homepage should immediately communicate:

- Thailand visa support
- simplicity
- trust
- fast communication
- modern professional service

The experience should feel:

- easy

- conversion-focused

# 2. Hero Section

## Goal

Communicate immediately:
- what the company does
- who it helps
- how to start

---

## Structure

Left side:
- headline
- supporting text
- CTA buttons
- trust indicators

Right side:
- subtle UI visual
OR
- refined Thailand visual
OR
- clean dashboard/process visual

- generic stock imagery
- tourist clichés
- beach visuals

- Thai visa support without the confusion.

## Supporting Copy

- visa types
- communication speed
